    NOTE: This installation is for C on VS Code using Linux

    1.

        sudo apt upgrade

        sudo apt autoremove
    2.
        sudo apt install gcc g++ clang gdb
    3. 
        Install VS Code extension: C/C++ Extension Pack
    4.
        Install VS Code extension: Code Runner

    Installation -dev complete
